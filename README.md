## Getting Started

To get a local copy up and running follow these simple steps.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/diogobruni/Spotifree.git
   ```
2. Install NPM packages
   ```sh
   npm install
   # or
   yarn
   ```
3. Run as develop environment
   ```sh
   npm run dev
   # or
   yarn dev
   ```
